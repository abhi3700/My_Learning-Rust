# Reqwest

## Overview

Reqwest provides a convenient, higher-level HTTP Client for Rust.

## Features

- Async and blocking Clients
- Plain bodies, JSON, urlencoded, multipart
- Customizable redirect policy
- HTTP Proxies
- Uses system-native TLS
- Cookies
